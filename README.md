# Trader Bot

A QQ bot with virtual stock trading plugin built on botpy framework.

## Quick Start

1. Clone this repo
2. pip install -r requirements.txt
3. cp bot_config.example.json bot_config.json
4. Edit bot_config.json with your appid and secret
5. python main.py

## Plugin

See plugins/trader/README.md for plugin documentation.
